# 2026-04-06 - nginx dev SSL hosting

## Summary
- Updated nginx config to add an HTTPS listener (`443`) using certificates generated by `scripts/make-dev-ssl-cert.sh`.
- Added HTTP-to-HTTPS redirect behavior so local traffic from `http://localhost:8080` lands on `https://localhost:8443`.
- Updated Docker Compose nginx service to publish host port `8443` and mount the repo `.ssl` directory into nginx cert paths.
- Updated top-level README with local HTTPS setup/run instructions.

## What Did Not Work
- Attempting to rely on TLS without mounting a cert directory was not viable because nginx requires certificate/key files at startup for the SSL server block.
